# Turing Pattern Generator (リアルタイム反応拡散シミュレータ)

これは、[チューリング・パターン（反応拡散モデル）](https://ja.wikipedia.org/wiki/%E5%8F%8D%E5%BF%9C%E6%8B%A1%E6%95%A3%E7%B3%BB)をシミュレーションし、シームレスなテクスチャを生成するインタラクティブなWebアプリケーションです。

シミュレーションと描画はWebGL (GLSL) によって実装されています。

## ✨ 主な機能

* **WebGLによるシミュレーション**: WebGL (GLSLシェーダー) を使用して、反応拡散の計算を実行します。

* **インタラクティブな操作**: マウスやタッチで画面をクリック/ドラッグすることで、シミュレーションに「種」（化学物質B）を追加し、パターンの成長を促すことができます。

* **シームレスなタイリング表示**: 生成された1024x1024のテクスチャは、シームレスにタイリング（繰り返し）表示されます。

* **パラメータのリアルタイム調整**: 以下のグレイ・スコットモデルのパラメータをスライダーで自由に調整し、パターンの変化を観察できます。

  * Feed Rate (F)

  * Kill Rate (k)

  * Diffusion Rate A (dA)

  * Diffusion Rate B (dB)

  * Time Step (dt)

* **プリセット**: 特徴的なパターンを生成するパラメータのプリセットが用意されています。

* **ダウンロード**: 現在のシミュレーション結果を1024x1024ピクセルのPNG画像としてダウンロードできます。
